This software is in alpha release. Please contact the authors before using in experiments or published work.

To install the latest development release of **fabricatr**, please ensure that you are running version 3.3 or later of R and run the following command:
`install.packages("fabricatr", repos=c("http://install.declaredesign.org", "https://cloud.r-project.org"), dependencies=TRUE)`
